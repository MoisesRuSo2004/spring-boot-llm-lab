# Spring AI + Groq Lab

Proyecto de laboratorio usando Spring Boot y Spring AI integrado con Groq (LLM).

## 🚀 Tecnologías
- Spring Boot
- Spring AI
- Groq API

## ⚙️ Configuración
Definir variable de entorno:

GROQ_API_KEY=tu_api_key

## ▶️ Ejecución
Ejecutar la clase:
Nivel2SpringaiApplication

## 🧪 Endpoints

POST /api/v2/chat  
POST /api/v2/chat/rapido  
GET /api/v2/chat/salud

## 📌 Notas
El proyecto usa Groq como proveedor LLM compatible con OpenAI.
